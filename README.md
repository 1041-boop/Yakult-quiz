# 商品知識クイズ｜兵庫ヤクルト販売

## 概要
兵庫ヤクルト販売スタッフ向けの商品知識学習クイズアプリです。

## 使い方
`index.html` をブラウザで開くだけで動作します。サーバー不要。

## カテゴリ
- 商品知識（乳製品）
- 商品知識（ジュース・清涼）
- 商品知識（化粧品）
- 商品知識（関連商品）
- 接客・販売知識
- その他

## 機能
- カテゴリ別・ランダム出題
- タイマー設定（カスタム秒数対応）
- ブックマーク機能
- XP・レベルシステム（Lv.1〜100）
- ダークモード・テーマカラー変更
- 管理者画面（問題の追加・編集・HTMLエクスポート）
- 100点で紙吹雪

## 注意
- 問題データはLocalStorageに保存されます
- AI生成機能はローカルファイルでは動作しません
- 新しいHTMLを開いた際、編集済みデータは保持されます
